# Django_Shortener_URL
Esta es un ejemplo sobre un Acortador de URL utilizando la librería PYSHORTENERS de Python

## Desafios

* Manejo de Rutas
* Manejo de Vistas
* Manejo de Formulario - Metodo GET
* Manejo de Librerias externas - PYSHORTENERS
